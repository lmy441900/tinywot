# TinyWoT Core

TinyWoT Core is the platform agnostic core of TinyWoT. It is still a work-in-progress.

TinyWoT is a [W3C Web of Things (WoT)](https://www.w3.org/WoT/) framework targetting resource-constrained embedded devices.
